
# ResourceR Web App
Web app for Riphah class resources 

## Features

1.  [x] Login and Signup
2. [x] Login with google
3.  [x] Role Base user account
4.  [x] Admin dashboard
5.  [x] Resources

	4. [x] Departments
	5. [x] Classes
	6. [x] Courses
	7. [x] Resources
	8. [x] Assignments
	9. [x] Labs
6. [x] Announcements 
7. [x] Themes
	6. [x] Dark
	7. [x] Light
8. [ ] Student Profile
9. [ ] Teacher Profile
10. [ ] Timetables
11. [ ] Notifications

	11. [ ] Class  
	12. [ ] Announcements
